# ACL-SIM
 Active Load Control for Flight Simulator.
 
 Pro-Sim and AC Servo controllers

Software:
- Prosim-AR 737
- Flight Simulator 2020 or P3D
- Custom ACL software control

Hardware:
- AC Servo Motors
- Arduino Mega
- RS232 Controller
